# Inicio de sesión y acceso

## Acceder al sistema

- Abre el enlace del sistema en tu navegador.
- Ingresa tu **usuario** y **contraseña**.

## Mensajes comunes

- **Usuario/contraseña inválidos**: verifica que no tengas mayúsculas activadas o espacios.
- **Bloqueo temporal**: el sistema puede bloquear intentos repetidos por seguridad. Espera unos minutos e intenta de nuevo.

## Requisitos de seguridad

- Si tu cuenta tiene **2FA activo**, luego de ingresar credenciales se te pedirá un segundo factor.

## Referencia técnica (para soporte)

- El flujo de autenticación está centralizado en:
  - `backend/core/auth-manager.php`
