# Perfil de usuario

## ¿Qué puedes gestionar en tu perfil?

Según el módulo instalado y tus permisos, normalmente podrás:

- Ver tu nombre de usuario.
- Cambiar datos básicos.
- Cambiar contraseña.
- Configurar métodos de seguridad como 2FA (si está habilitado por el sistema).

## Recomendaciones

- Mantén tu correo actualizado si el sistema usa correo para recuperación/2FA.

## Referencia técnica (para soporte)

- Autenticación y sesión:
  - `backend/core/auth-manager.php`
- 2FA:
  - `backend/core/auth-2fa-manager.php`
  - `backend/core/recovery-manager.php`
