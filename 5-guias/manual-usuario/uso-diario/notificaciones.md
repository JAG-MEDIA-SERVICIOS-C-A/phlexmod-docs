# Notificaciones (tiempo real)

## Qué son

El sistema puede mostrar notificaciones en tiempo real (por ejemplo: avisos, confirmaciones o eventos del sistema).

## Si no aparecen notificaciones

- Recarga la página.
- Verifica tu conexión.
- Si el problema persiste, el servidor WebSocket puede estar inactivo.
